Supports multiverse multiple world teleports

Allows signs to be created that may be right clicked to teleport to the destination..

The sign must follow this format:

Waypoint:
<destination>

where destination is the name of the waypoint you wish to teleport to

Permissions are:
Either of these permissions will enable the use of sign teleporting.
waypoints.go
waypoints.signs.go

This permission will allow users to make waypoint signs.
waypoints.signs.create
